Help

The Search Form allows you to search fro either genes or protein stings of interest.

Gene Search

To search for a particular gene, please enter an Ensembl gene identifier or gene symbol in the box below "Enter a term to search for a gene of interest." and click the "Gene Search" button.

Protein Search

To search for a particular protein string or strings, please enter each search term on a separate line in the box below "Enter protein search strings.". Click the "Protein Search" button to begin the search.
  • The "*" character will match any number of characters between the surrounding characters
  • The "?" character will match a single character between the surrounding characters
  • For an exact match, check the "Exact match" box
